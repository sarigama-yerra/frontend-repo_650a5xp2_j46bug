# GoldenPlatter Catering — React + Tailwind UI

A modern, animated, fully responsive static website UI for a premium catering company. Built with React (Vite), Tailwind CSS, and Framer Motion for smooth entrance and hover animations.

Features
- Elegant, appetizing design using warm golds, creams, and greens
- Sticky navbar that becomes solid on scroll
- Animated hero with parallax illustration and CTA buttons
- About, Services, Gallery (masonry), Testimonials (auto-sliding), CTA banner, and Footer
- Glassmorphism cards, gradients, and subtle hover effects
- Fully responsive for desktop, tablet, and mobile
- No backend or forms — placeholder content throughout

Tech
- React + Vite
- Tailwind CSS
- Framer Motion
- Lucide Icons

Getting Started
1. Install dependencies
   npm install

2. Start the dev server (http://localhost:3000)
   npm run dev

3. Build for production
   npm run build

Customization
- Replace placeholder texts in the section components
- Swap gallery images and testimonial content
- Update brand name/colors in the navbar and footer
- You can replace the Spline scene in the hero with a Lottie or SVG illustration if preferred

Notes
- Fonts: Playfair Display for headings, Inter for body text (loaded via Google Fonts)
- Animations use Framer Motion. Example entrance animations are implemented in Hero and About sections.
